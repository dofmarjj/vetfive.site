# Vetfive — ветеринарна клініка

Статичний сайт (HTML / CSS / JS).

## Структура

- `index.html` — головна
- `poslugy.html` — послуги
- `doctors/` — сторінки лікарів
- `assets/` — зображення
- `styles.css`, `script.js` — стилі та скрипти

## Локальний перегляд

Відкрий `index.html` у браузері або підніми простий сервер:

```bash
npx serve .
```

## GitHub Pages

Налаштування: **Settings → Pages → Branch `main`, folder `/ (root)`**.

Сайт буде за адресою: `https://<нік>.github.io/<назва-репо>/`

У цьому випадку перевір шляхи до сторінок лікарів і послуг (відносні посилання вже від кореня репо).
